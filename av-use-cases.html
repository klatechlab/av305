ภาพรวมรูปแบบการใช้งาน

ระบบภาพห้องประชุม 9 รูปแบบ

หน้านี้ใช้เป็นแผนที่เลือกแนวทางการใช้งานก่อนเปิดคู่มือรายละเอียด ครอบคลุมงานประชุมภายในห้อง งาน Hybrid การจัดภาพด้วย NovaStar VX600 การส่ง Program Feed ไปยัง Zoom การบันทึก และการกระจายภาพไปยังพื้นที่อื่น

หลักสำคัญ

ตำแหน่ง Input และ Output ในแต่ละรูปแบบเป็นตัวอย่างการติดตั้ง สามารถเปลี่ยน Source หรือย้ายพอร์ตตามหน้างานได้ รูปแบบทั้ง 9 แบบจึงเป็น “แนวทางการใช้งาน” ไม่ใช่ผังการเสียบสายแบบตายตัว

3 กลุ่มการใช้งาน

เลือกกลุ่มก่อน แล้วค่อยเลือกรูปแบบ

A

ระบบพื้นฐานและ Hybrid ทั่วไป

รูปแบบที่ 1–3 เหมาะกับงานประชุมและงานบรรยายทั่วไป

B

การจัดภาพด้วย NovaStar VX600

รูปแบบที่ 4–7 ใช้ Main Layer, PIP, Preset และ Monitor Out

C

บันทึกและกระจาย Program Feed

รูปแบบที่ 8–9 นำภาพผลลัพธ์ไปใช้ต่อกับ OBS, Recorder และห้องเสริม

กลุ่ม A

ระบบพื้นฐานและ Hybrid ทั่วไป

รูปแบบที่ 1–3
1 ง่าย

การนำเสนอในห้องแบบปกติ

Local Presentation

เลือกภาพจาก Maxhub, HDMI เวที หรือกล้อง PTZ ผ่าน VAVE แล้วกระจายภาพไปยัง TV, Projector และ LED Wall โดยไม่มี Zoom

เหมาะกับ

การประชุมภายใน การอบรม และการบรรยายทั่วไป

เปิดคู่มือระบบหลัก →
2 ใช้งานจริงแล้ว

คนออนไลน์ขึ้นจอ LED Wall

Remote Participants on LED Wall

กล้องในห้องเข้า Capture Card และ Zoom ส่วน Notebook ส่งภาพผู้เข้าร่วมออนไลน์กลับเข้า VAVE เพื่อขึ้น LED Wall

คนในห้องเห็น

Gallery View หรือ Speaker View ของผู้เข้าร่วมออนไลน์

เปิดคู่มือ Hybrid →
3 น่าทดลอง

คนออนไลน์เห็นวิทยากรบนเวที

Stage Camera to Online

วิทยากรไม่ต้องเข้า Zoom เอง ทีมเทคนิคใช้กล้องถ่ายวิทยากร แล้วส่งผ่าน Capture Card เข้า Notebook และ Zoom

เหมาะกับ

สัมมนา การบรรยาย และงานที่ทีมเทคนิคควบคุม Zoom ให้ทั้งหมด

กลุ่ม B

การจัดภาพด้วย NovaStar VX600

รูปแบบที่ 4–7
4 ระดับกลาง

คนออนไลน์เห็นภาพเดียวกับ LED Wall

LED Wall Program to Online

NovaStar จัดภาพและส่งขึ้น LED Wall จากนั้นใช้ HDMI Monitor Out ผ่าน Capture Card เข้า Zoom เพื่อให้คนออนไลน์เห็น Program Feed ใกล้เคียงกับภาพในห้อง

อ่านเรื่อง Monitor Out →
5 ระดับกลาง

สไลด์หลักพร้อมภาพวิทยากร PIP

Presentation with Speaker PIP

ใช้สไลด์เป็น Main Layer และวางกล้องวิทยากรเป็น PIP ภาพเดียวกันสามารถส่งไป LED Wall และส่งออกทาง Monitor Out เข้า Zoom

6 ต้องวางระบบ

วิทยากรออนไลน์ร่วมกับสไลด์

Remote Speaker with Presentation

จัดภาพวิทยากรออนไลน์กับสไลด์บน NovaStar โดยเลือกได้ว่าจะให้วิทยากรหรือสไลด์เป็นภาพหลัก

แนะนำใช้ Notebook แยกหน้าที่ เพื่อป้องกันภาพ Zoom วนกลับเข้าตัวเอง

7 ซับซ้อนกว่า

เสวนา Hybrid หลายภาพ

Hybrid Panel Discussion

ใช้ Main Layer และ PIP เพื่อแสดงผู้ดำเนินรายการ วิทยากรในห้อง และวิทยากรออนไลน์ร่วมกัน เหมาะกับ Layout ที่เตรียมไว้ล่วงหน้า

กลุ่ม C

บันทึกและกระจาย Program Feed

รูปแบบที่ 8–9
8 ระดับกลาง

บันทึกและถ่ายทอด Program Feed

Program Recording and Streaming

ส่ง NovaStar Monitor Out เข้า Capture Card แล้วนำไปใช้กับ OBS, Zoom, Recorder หรือระบบถ่ายทอดสด

ได้ภาพแบบใด

ภาพที่ผ่านการจัด Layout, PIP และการสลับ Source แล้ว

9 ใช้งานได้จริง

ส่งภาพไปห้องเสริมหรือจอควบคุม

Overflow and Control Monitor

กระจาย Program Feed ไปยังจอทีมเทคนิค ห้องพักวิทยากร ห้องสำรอง หรือพื้นที่รองรับผู้เข้าร่วมเพิ่มเติม

หากวน Monitor Out กลับเข้า VAVE ต้องหลีกเลี่ยงการส่ง Monitor Return กลับออกทาง Output A

สรุปเปรียบเทียบทั้ง 9 รูปแบบ

ลำดับ รูปแบบ จุดเน้น ความซับซ้อน
